# Privacy Gecko Organization Profile

This repository contains the organization profile README that appears on the [Privacy Gecko GitHub page](https://github.com/PrivacyGecko).

## Structure

```
.github/
├── README.md           # This file - explains the repo
└── profile/
    └── README.md       # Organization profile (displays on org page)
```

## Updating the Profile

To update the organization profile:

1. Edit `profile/README.md`
2. Commit and push changes
3. GitHub will automatically update the org page within a few minutes

## Customization

**Adding Images:**
Store images in this repo and reference with absolute URLs:
```markdown
![Image](https://github.com/PrivacyGecko/.github/blob/main/profile/assets/image.png)
```

**Dynamic Badges:**
Use shields.io for live statistics:
```markdown
![Stars](https://img.shields.io/github/stars/PrivacyGecko/gecko-advisor?style=social)
```
